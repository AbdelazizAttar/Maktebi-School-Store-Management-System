# 📚 Maktebi – School Store Management System

Maktebi is a comprehensive full-stack web application developed to modernize and simplify the management of school stores, bookstores, and educational supply shops. The platform combines a complete Point of Sale (POS) system with inventory management, sales tracking, financial reporting, barcode integration, and user administration in a single intuitive solution.

Designed and developed as a personal university project, Maktebi focuses on improving operational efficiency by automating inventory control, sales processing, customer transactions, and business analytics. The application provides administrators and cashiers with a secure, responsive, and user-friendly interface for managing every aspect of a school store.

# 🎯 Project Objectives

The main objective of Maktebi is to replace traditional manual store management with a centralized digital platform capable of handling inventory, sales, reporting, and administrative tasks in real time.

The system helps administrators and cashiers efficiently manage products, monitor stock levels, process customer purchases, generate invoices, and analyze business performance through an interactive dashboard.

# 💻 Installation

## Requirements

* PHP 8+
* MySQL / MariaDB
* Apache (XAMPP, WAMP or Laragon)

## Clone the repository

```bash
git clone https://github.com/AbdelazizAttar/Maktebi-School-Store-Management-System.git
```

## Configure the database

1. Create a MySQL database.
2. Import the provided SQL file.
3. Update the database credentials inside `config.php`.

## Start the application

Run Apache and MySQL, then open:

```
http://localhost/Maktebi-School-Store-Management-System/
```
